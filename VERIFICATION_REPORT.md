## 🔍 Known Limitations & Solutions

### 1. LeetCode GraphQL Parser
- **Issue**: acSubmissionNum sometimes returns empty array/string
- **Impact**: LC stats might show "N/A" occasionally
- **Solution**: LeetCode API is inconsistent; fallback to null is acceptable
- **Workaround**: Can add retry logic with exponential backoff

### 2. Codeforces Rate Limiting
- **Issue**: API allows ~1 request/second per IP
- **Impact**: Heavy users might get rate-limited
- **Solution**: Already implemented 30-min caching; sequential requests queued
- **Workaround**: Implement request queue in codeforcesService.js

### 3. Free MongoDB Tier Limits
- **Issue**: 512MB storage, no multi-region replication
- **Impact**: Can't scale to thousands of users without upgrade
- **Solution**: Use paid tier ($57+/month) for production
- **Workaround**: Implement data archival strategy for old goals/notes

### 4. Email Reminders (Optional)
- **Issue**: Requires Gmail app password (not account password)
- **Solution**: Setup documented in DEPLOYMENT.md
- **Status**: Not tested (requires Gmail setup), but code is ready
